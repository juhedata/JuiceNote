# JuiceNote 聚思笔记本

一个更适合软件开发者的笔记本。

## Features

🎯 支持大量富文本组件   

📦 支持直接在笔记本内运行代码 

## 开发

- 确认环境

  以下环境测试通过：

  - node v16.17.0
  - npm 8.15.0

- 项目初始化

```sh
npm ci
npm install node-gyp@latest -g
npm run rebuild
```

- 运行与调试

```
npm run dev
```

- 查看样式

```
npm run t
```
